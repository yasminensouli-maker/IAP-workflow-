📊 Cost Explorer Guide — How to Pull the Baseline & POP
Step-by-step · Customer sends directly to TCC and Intel · AWS cannot forward this data
▼
Step-by-Step: Pulling the Baseline
  1. AWS Console → Billing & Cost Management → Cost Explorer
  2. Set Service filter to EC2
  3. Group by Instance Type
  4. Set date range to last 3–12 months
  5. Filter to Intel families: m8i, c8i, r8i, x8i only
    ⚠ Exclude Trainium (Trn) and Graviton (Arm)
  6. Save this filter as a named report — reuse each quarter
  7. Export as CSV or screenshot
  8. Customer emails export directly to TCC and Intel
Quarterly POP — Same Steps, New Date Range
💡 Pro Tip: Save the Filter
When pulling the baseline, save the Cost Explorer filter configuration as a named saved report. For each quarterly POP, the customer reopens the same report, updates only the date range, and exports. A 5-step setup becomes a 1-click action every quarter.
